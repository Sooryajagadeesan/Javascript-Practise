Introduction to Javascript

Javascript is the primary scripting language for HTML.

We can think of HTMl, CSS, JS as the 3 primary pillars of front-end Web development. We can come to a conclusion that,

  • HTML : Defines the Content of the web page
  • CSS : Defines the Layout for Content of the web page
  • JS : Defines the Behaviour(functionality) of Content of the web page

What can we do with Javascript?

  • We can create Dynamic Pages.
  • We can change the web content using DOM methods.
  • We can handle events.
  • We can give the page an asynchronous behaviour.
